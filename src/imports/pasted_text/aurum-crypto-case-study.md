AURUM Crypto Exchange

UX/UI Design Case Study

Description

AURUM is a cryptocurrency exchange platform designed to simplify digital asset management and currency conversion.
The goal of the project was to create a clear and secure interface that allows users to exchange assets and manage transactions with confidence.

PROJECT INFO

Role
UX/UI Designer

Platform
Web & Mobile

Industry
FinTech / Crypto

Scope

• Product interface design
• Exchange flow UX
• Dashboard UI
• Mobile adaptation

PROBLEM

Cryptocurrency platforms often overwhelm users with complex financial data and unclear transaction flows.

Many users struggle with:

• understanding balances
• navigating exchange interfaces
• confirming financial actions safely

The challenge was to design an interface that simplifies crypto exchange while maintaining trust and clarity.

GOAL

Create a cryptocurrency exchange interface that:

• simplifies asset conversion
• clearly displays financial data
• minimizes user mistakes during transactions
• works smoothly on both desktop and mobile

LANDING EXPERIENCE

The landing page introduces the platform and communicates the core value of the product: simple and secure cryptocurrency exchange.

The interface focuses on clarity and trust through:

• structured layout
• minimal visual noise
• clear feature highlights

(აქ ჩასვი landing desktop + mobile)

VISUAL DIRECTION

The visual style uses a dark interface, which is common for financial dashboards.

Dark UI improves data readability and allows users to focus on numbers and transactions.

The accent color highlights important actions such as exchange confirmation and navigation elements.

Design principles:

• clarity
• trust
• minimal distraction
• strong visual hierarchy

(აქ ჩასვი dark + light screens)

DASHBOARD

The dashboard is the main workspace where users manage their assets.

Key components:

• real-time balance display
• exchange module
• asset selection
• quick transaction actions

The layout prioritizes the exchange action while keeping financial data visible.

(აქ ჩასვი exchange dashboard)

EXCHANGE INTERACTION

Currency conversion is the core action of the platform.

The exchange interface allows users to:

• choose currencies
• view exchange values instantly
• confirm transactions quickly

Clear hierarchy ensures users always understand what they are exchanging and receiving.

(აქ ჩასვი dropdown interaction GIF)

TRANSACTION FLOW

Financial operations require confirmation to prevent mistakes.

Exchange process:

enter exchange amount

confirm transaction

verify wallet information

This flow ensures users clearly review their actions before completing a transaction.

(აქ ჩასვი confirm modal + wallet modal)

ERROR HANDLING

Financial interfaces must clearly communicate issues.

Error states were designed to:

• explain the problem clearly
• guide users to fix it
• prevent incorrect transactions

Clear feedback improves user confidence and reduces mistakes.

(აქ ჩასვი error modal)

TRANSACTION HISTORY

Users need quick access to their financial activity.

The transaction history interface allows users to:

• view completed transactions
• track statuses
• filter records

This helps users easily monitor financial activity and manage their assets.

(აქ ჩასვი transaction table)

MOBILE EXPERIENCE

The mobile interface was designed to keep the core exchange functionality accessible on smaller screens.

Key priorities:

• simplified navigation
• readable financial data
• quick exchange access

The goal was to maintain the same clarity and efficiency as the desktop interface.

(აქ ჩასვი mobile screen)

RESULT

The final design delivers a modern cryptocurrency exchange platform that focuses on usability and trust.

The interface simplifies complex financial actions and allows users to manage digital assets quickly and confidently.
